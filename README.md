BurgerStack ERP 개발

# 🍔 Burger Stack ERP

버거 프랜차이즈 매장의 재고, 발주, 문의, 점포 운영을 통합 관리하기 위한 ERP 시스템입니다.

---

## 📌 프로젝트 소개

Burger Stack ERP는 버거 프랜차이즈 운영 환경을 가정하여 제작한 ERP 프로젝트입니다.

점주와 본사 관리자를 구분하여:

- 재고 관리
- 식자재 발주
- 점포 운영
- 문의 및 공지사항 관리
- 재고 반영 요청
- 통계 및 대시보드

등의 기능을 제공합니다.

---

## 🎯 프로젝트 목표

- 점포별 재고 현황 통합 관리
- 발주 프로세스 자동화
- 본사/점주 역할 기반 시스템 구축
- ERP 실무 흐름 반영
- 재고 누락 및 폐기율 관리 효율화

---

## 👨‍💻 주요 기능

### 🔐 사용자 인증

- 로그인
- 아이디 / 비밀번호 찾기
- 권한별 접근 제어

- 점포 상태 관리

### 📦 재고 관리

- 점포별 재고 조회
- 전산재고 / 실재고 비교
- 재고 차액 계산
- 파일 업로드 기반 재고 반영 요청
- 엑셀 / PDF Export

### 🛒 발주 관리

- 식자재 발주 신청
- 발주 승인 / 반려
- 발주 상태 관리
- 발주 상세 조회
- 발주 이력 관리

### 📢 공지사항

- 공지사항 CRUD
- 조회수 기반 목록 조회

### ❓ 문의사항

- 문의 등록
- 문의 답변
- 문의 상태 관리

### 📊 대시보드

- 실시간 매출 현황
- 부족 재고 TOP5
- 폐기율 통계
- 최근 발주 요청
- 최근 문의사항
